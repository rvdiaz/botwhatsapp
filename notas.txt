para crear el package.json usamos el comando dentro del directorio del proyecto "npm init -y"
para compilar por ejemplo el index.js ponemos en la terminal "node index.js"
para que la consola compile sola al hacer algun cambio instalamos "npm i nodemon -g"
para compilar el nodemon hacemos "nodemon index.js", para cancelar la compilacion "control + c"
en los script del package creamos una variable y la sustituimos por el comando que queramos y luego pondria en la consola "npm run {variable del script}"


para instalar express seria "npm i express"

para que el node coja las variables de entorno agrago el comando "npm i dotenv"

para validar formularios se instala "npm i express-validator"

para utilizar la base de datos mongo y ligarla con node instalamos mongoose con el comando "npm i mongoose"

para encriptar contrase;as se utiliza el "npm i bcryptjs"

para generar un token utilizamos "npm i jsonwebtoken"

para hacer las cors utilizar "npm install cors"

para validar fechas instalo moment "npm install moment"
